## 📋 プロジェクト概要

**学ナビ**は、就活早期化に対応した学生支援サービスのランディングページです。
28・29卒（大学1・2年生）をターゲットに、無料相談会やイベント情報を提供します。

## 🛠️ カスタマイズ方法

### PDFを追加・変更したい
1. PDFファイルを `pdfs/` フォルダに保存
2. `index.html` のカルーセル部分を編集：

```html
<div class="swiper-slide">
    <div class="pdf-card" data-pdf-url="pdfs/新しいPDF.pdf" data-title="タイトル">
        <div class="pdf-thumbnail-container">
            <canvas class="pdf-thumbnail" data-pdf-url="pdfs/新しいPDF.pdf"></canvas>
            <div class="pdf-loading">
                <div class="pdf-loading-spinner"></div>
                <p>読み込み中...</p>
            </div>
        </div>
        <div class="pdf-card-content">
            <h3 class="pdf-card-title">タイトル</h3>
            <p class="pdf-card-date">日付</p>
            <p class="pdf-card-description">説明</p>
            <button class="pdf-card-btn">詳細を見る</button>
        </div>
    </div>
</div>
```

### 画像を変更したい
1. 新しい画像を `images/` フォルダに保存
2. `index.html` のimgタグを更新：

```html
<img src="images/新しい画像.jpg" alt="説明" class="illustration-warm">
```

### 配色を変更したい
`css/style.css` の `:root` セクション：

```css
:root {
    --primary-deep-purple: #4B2E83;  /* ← 好きな色に変更 */
    --base-mint-green: #63D1C6;
    --sub-coral-orange: #FF8A65;
    --accent-vivid-yellow: #F4E04D;
}
```

### autoplayの間隔を変更したい
`js/main.js` のSwiper設定：

```javascript
autoplay: {
    delay: 5000, // ← 5秒に変更（ミリ秒単位）
    disableOnInteraction: false,
},
```

### フォーム送信先を設定したい
`js/main.js` の送信処理部分：

```javascript
// 送信処理（実際のAPI連携はここで実装）

    await fetch('/api/contact', {
```

## 🐛 トラブルシューティング

### PDFが表示されない
**原因**: ブラウザのローカルファイル制限  
**対処**: ローカルサーバーで開く（方法2）

### Swiperが動かない
**原因**: JavaScriptの読み込みエラー  
**対処**: コンソールでエラーを確認。ファイルパスが正しいか確認。

### 画像が表示されない
**原因**: ファイルパスが間違っている  
**対処**: `images/` フォルダにファイルが存在するか確認。

### フォームが送信されない
**原因**: バリデーションエラー  
**対処**: すべての必須項目を入力。コンソールでエラーを確認。
